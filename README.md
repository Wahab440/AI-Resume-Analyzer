# AI-Resume-Analyzer
AI Resume Analyzer is a full-stack web application that analyzes a user's resume and extracts important skills using Artificial Intelligence. The system allows users to upload a PDF resume, after which the AI model processes the document, detects relevant technical skills, and generates a resume score based on the identified skills.

The project uses a MERN-based architecture with a Python AI model for resume analysis. The frontend is built with React, the backend uses Node.js and Express for API handling, and the AI component is implemented in Python to perform text extraction and skill detection from resumes.

This project demonstrates the integration of AI with a modern web stack to build a practical application that can assist users in evaluating and improving their resumes.
